# /resolve-conflicts — Resolve Git Conflicts

Resolve git merge, rebase, or cherry-pick conflicts. Escalates ambiguous conflicts rather than guessing.

## Arguments
- `$ARGUMENTS` — Optional: specific file or context (e.g., "src/auth.ts", "rebase onto main"). If omitted, resolves all conflicts in the current repo.

## Instructions

Use the `resolve-conflicts` skill to execute this command.
