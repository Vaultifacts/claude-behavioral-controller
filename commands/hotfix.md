# /hotfix — Production Hotfix Workflow

Branch from a release tag, apply a minimal fix, and fast-track to merge. Enforces merge-back to main and semver patch increment.

## Arguments
- `$ARGUMENTS` — Description of the issue to fix (e.g., "null pointer in payment handler", "v2.3.1 login crash").

## Instructions

Use the `hotfix` skill to execute this command.
