# /perf-profile — Performance Profile

Profile and diagnose performance issues — bundle size, runtime CPU/memory hotspots, slow queries, and render performance.

## Arguments
- `$ARGUMENTS` — Optional: specific area or file to profile (e.g., "bundle", "memory", "db", "src/api"). If omitted, profiles the full project.

## Instructions

Use the `perf-profile` skill to execute this command.
