# /dep-audit — Dependency Audit

Audit project dependencies for outdated packages, known CVEs, and version drift. Produces a prioritized update plan — does not auto-update without confirmation.

## Arguments
- `$ARGUMENTS` — Optional: specific ecosystem or package (e.g., "npm", "python", "lodash"). If omitted, audits all detected package managers.

## Instructions

Use the `dep-audit` skill to execute this command.
