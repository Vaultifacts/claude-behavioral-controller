# /run-tests — Run Tests and Fix Failures

Run the test suite, diagnose failures, and fix them. Covers the run → fail → fix → verify loop with a 3-iteration cap.

## Arguments
- `$ARGUMENTS` — Optional: specific test file, test name, or ecosystem (e.g., "auth.test.ts", "test_login", "python"). If omitted, runs the full suite.

## Instructions

Use the `run-tests` skill to execute this command.
